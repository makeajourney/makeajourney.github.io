### tensorflow

tensor  
placeholder  
variable  
rank: Number of dimensions  
- 0: scalar
- 1: vector
- 2: matrix
- 3: n-Tensor or n dimensional Tensor

shape: Number of elements in each dimension. It explains structure of tensor.
